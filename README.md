# go-backup 💾

ブラウザで操作できるローカルファイルバックアップツールです。

## 機能

- 📦 **今すぐバックアップ** — フォルダを指定してワンクリックでバックアップ
- ⏰ **自動バックアップ** — 毎日指定した時刻に自動実行
- 📋 **履歴管理** — バックアップ結果を最大20件記録
- 🔄 **リアルタイム更新** — 実行中の状態がブラウザに反映

## 使い方

```bash
git clone https://github.com/yourusername/go-backup.git
cd go-backup
go run cmd/main.go
```

ブラウザで `http://localhost:8080` を開く。

## オプション

```bash
go run cmd/main.go --port 9090  # ポート変更
```

## バックアップの仕組み

バックアップ先に `backup_2024-01-15_09-00-00` のようなタイムスタンプ付きフォルダが作られます。

## セキュリティ

- 外部ライブラリを一切使用していません
- インターネットへの通信は一切行いません
- 動作はすべてローカル（自分のPC内）で完結します
- 不安な方はソースコードをご自身で確認できます

外部通信がないことを確認するコマンド：
```cmd
findstr /r /s "http.Get\|http.Post\|net.Dial" internal\*.go cmd\*.go
```
何も表示されなければ外部通信なしの証明です。

## ウイルススキャン結果

VirusTotalスキャン: **71社中2社が誤検知**

Goで作られたexeは構造上、一部のウイルス対策ソフト（Microsoft DefenderのWacatacB!mlなど）が誤検知することがあります。これはGoのバイナリ構造がマルウェアに似ていると判定されやすいためで、Go製ツール全般に起きる既知の問題です。

**そのため、このツールはexeではなくソースコードでの配布を推奨しています。**
ご自身の環境で `go run cmd/main.go` を実行することで、exeを一切使わずに動作します。
ソースコードはGitHubで全公開しており、動作の透明性を保証しています。
