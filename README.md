# go-backup 💾

ブラウザで操作できるローカルファイルバックアップツールです。

## 機能

- 📦 **今すぐバックアップ** — フォルダを指定してワンクリックでバックアップ
- ⏰ **自動バックアップ** — 毎日指定した時刻に自動実行
- 📋 **履歴管理** — バックアップ結果を最大20件記録
- 🔄 **リアルタイム更新** — 実行中の状態がブラウザに反映

## 使い方

```bash
git clone https://github.com/yourusername/go-backup.git
cd go-backup
go run cmd/main.go
```

ブラウザで `http://localhost:8080` を開く。

## オプション

```bash
go run cmd/main.go --port 9090  # ポート変更
```

## バックアップの仕組み

バックアップ先に `backup_2024-01-15_09-00-00` のようなタイムスタンプ付きフォルダが作られます。
